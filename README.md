# 🌐 Принципи роботи комп'ютерних мереж та мережева безпека

Навчальний модуль для 10-11 класів (Поглиблений рівень)

> **Про проект:** Даний репозиторій містить матеріали курсу, розроблені в рамках лабораторної роботи з дисципліни "ШІ в освіті". Контент, тести та ілюстрації створені за допомогою інструментів генеративного штучного інтелекту (LLM, DALL-E 3 / Midjourney).

---

## 📚 Навігація по курсу

### 📖 Блок 1: Теорія
Тут зібрані фундаментальні знання про те, як працює інтернет під капотом, та як хакери намагаються його зламати.
* [**Основний теоретичний матеріал**](theory/main-content.md) — Моделі мереж, адресація, криптографія та Zero Trust.
* [**Аналіз реальних кейсів**](theory/examples.md) — Розбір HTTP-запитів та зберігання паролів.
* [**Глосарій термінів**](theory/glossary.md) — Словник інженера з кібербезпеки.

### 🛠 Блок 2: Практика
У цьому блоці ми переходимо від теорії до написання коду та використання професійних утиліт.
* [**Практичні завдання (CLI та Crypto)**](practice/tasks.md) — Мережева розвідка та робота з хешами в CyberChef.
* [**Лабораторні роботи (Python та Wireshark)**](practice/labs.md) — Створення власного TCP-сервера та аналіз дампів трафіку.
* [**Ситуаційні проекти**](practice/projects.md) — Проектування безпечної мережі та аналіз фішингу, згенерованого ШІ.

### 🎯 Блок 3: Оцінювання
Перевірте, чи готові ви до роботи в реальному ІТ-середовищі.
* [**Тести для самоперевірки (15 питань)**](tests/self-check.md) — Тренажер для закріплення бази.
* [**Підсумкове оцінювання**](tests/assessment.md) — Три складні ситуаційні задачі без готових відповідей.

### 🔗 Блок 4: Додаткові ресурси
* [**Відео та рекомендована література**](resources/links.md) — Профільна література та відео лекції для поглиблення знань.
